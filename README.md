🌈 ChromaForm – Interactive Registration Hub

A modern and visually stunning Interactive Registration Form built using HTML, CSS, and JavaScript. ChromaForm combines beautiful glassmorphism design, animated backgrounds, real-time form validation, dynamic progress tracking, and responsive user interactions to deliver an engaging registration experience.

🚀 Project Overview

ChromaForm is designed to demonstrate advanced frontend development concepts through an interactive registration system. The application provides instant feedback, dynamic form completion tracking, password visibility controls, dark/light theme switching, animated particles, and responsive design.

✨ Features
🎨 Ultra-colorful Glassmorphism UI
🌈 Animated Gradient Background
✨ Floating Particle Effects
📋 Real-Time Form Completion Progress Bar
🔒 Password Visibility Toggle
